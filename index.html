HTML5 is a markup language used for structuring and presenting content for the World Wide Web.

It is both the latest revision of and the new standard for HTML. Web browsers can read HTML files and compose them into visible or audible web pages. Browsers do not display the HTML tags and scripts, but use them to interpret the content of the page. HTML describes the structure of a website semantically along with cues for presentation, making it a markup language, rather than a programming language.

Cascading Style Sheet 3 (CSS3) is a style sheet language used for presentation and layout.

CSS is designed primarily to enable the separation of document content from document presentation, including elements such as the layout, colors, and fonts.[2] This separation can improve content accessibility, and provide more flexibility and control in the specification of presentation characteristics. It is used to describe the look and formatting of documents written in markup languages.

Ruby on Rails, or simply Rails, is an open source web application framework written in Ruby.

Ruby on Rails is a breakthrough in lowering the barriers of entry to programming. Powerful web applications that formerly might have taken weeks or months to develop can be produced in a matter of days. It allows you to write less code while accomplishing more than many other languages and frameworks.

JavaScript (JS) is a simple, powerful, and popular programming language that is built into web browsers.

JavaScript is a lightweight, interpreted, object-oriented language with first-class functions. Learning JavaScript is especially useful if you are a web designer and already know HTML and CSS, because it is used to make web pages interactive. However, JavaScript is not limited to making interactive web pages; you can also use it for server-side programming using a framework like Node.js.

Bubble is a general-purpose system for creating web and mobile apps.

Bubble is designed to replace programming languages and frameworks traditionally used for web development. It replaces back-end technologies such as Ruby on Rails, Python / Django, and PHP, and it replaces front-end technologies such as HTML, CSS, and Javascript. Unlike those technologies, Bubble makes some assumptions about what you want to build. Specifically, it assumes you're building a multi-user interactive application for web or mobile.
